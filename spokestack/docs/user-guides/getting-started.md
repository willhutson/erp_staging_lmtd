# Getting Started with SpokeStack

Welcome to SpokeStack! This guide will help you get up and running quickly.

---

## First Login

1. **Access the Platform**
   - Go to your organization's SpokeStack URL (e.g., `yourcompany.spokestack.io`)
   - Click "Sign In" and enter your credentials

2. **Explore the Hub**
   - The Hub is your central dashboard showing:
     - Quick access to all modules
     - Recent activity
     - Pending tasks
     - Team announcements

3. **Set Up Your Profile**
   - Click your avatar in the top-right corner
   - Select "Settings" > "Profile"
   - Add your photo, contact info, and preferences

---

## Navigation

### Sidebar
The left sidebar provides access to all modules:
- **Hub** - Your main dashboard
- **ERP** - Briefs, Time, Leave, Team, RFP
- **Agency** - Clients, Projects, Resources
- **Marketing** - Analytics, Trackers, Builder
- **Admin** - Settings and management (if you have access)

### Header
The top header includes:
- **Search** - Find anything across the platform
- **SpokeChat** - Team messaging (click the megaphone icon)
- **Notifications** - Alerts and updates
- **Profile** - Settings and sign out

---

## Common Tasks

### Submitting a Brief
1. Navigate to **Briefs** from the sidebar
2. Click **New Brief** button
3. Select the brief type (Video, Design, Copy, etc.)
4. Fill in the required fields
5. Click **Submit**

### Logging Time
1. Navigate to **Time Tracking**
2. Click **Add Entry** or use the timer
3. Select the project/client
4. Enter hours and description
5. Save your entry

### Requesting Leave
1. Navigate to **Leave**
2. Click **Request Leave**
3. Select dates and leave type
4. Add notes if needed
5. Submit for approval

### Using SpokeChat
1. Click the megaphone icon in the header
2. Select a channel or start a DM
3. Type your message and send
4. Use @mentions to notify team members

## View Options

Many modules offer multiple view modes:

| View | Best For |
|------|----------|
| **List** | Detailed information, sorting, filtering |
| **Kanban** | Visual workflow, drag-and-drop status changes |
| **Timeline/Gantt** | Deadline tracking, resource planning |

Switch views using the toggle buttons at the top of each module.
