# SpokeChat Guide

SpokeChat is SpokeStack's built-in team communication tool. It provides real-time messaging, channels, and announcements.

---

## Accessing SpokeChat

**Quick Access**: Click the megaphone icon in the header
**Full View**: Navigate to SpokeChat from the sidebar

The megaphone icon shows a badge with unread message count.

---

## Features

### Channels
Team-wide or topic-specific conversations.

| Channel Type | Description |
|--------------|-------------|
| **#general** | Company-wide announcements |
| **#team-[name]** | Department-specific channels |
| **#project-[name]** | Project-focused discussions |
| **#random** | Casual conversation |

### Direct Messages (DMs)
Private one-on-one or group conversations.

### Announcements
Important messages that require acknowledgment.

---

## Sending Messages

1. Select a channel or DM conversation
2. Type your message in the input box
3. Press Enter or click Send

### Formatting
- **Bold**: `*text*` or Ctrl+B
- *Italic*: `_text_` or Ctrl+I
- `Code`: `` `text` ``
- > Quote: `> text`

### Mentions
- **@username** - Notify a specific person
- **@channel** - Notify everyone in the channel
- **@here** - Notify active members only

---

## Managing Notifications

### Notification Settings
1. Click the gear icon in SpokeChat
2. Adjust per-channel settings:
   - All messages
   - Mentions only
   - Muted

### Do Not Disturb
Enable DND mode to pause all notifications.

---

## File Sharing

Drag and drop files into the message box to share:
- Images (displayed inline)
- Documents (PDF, Word, Excel)
- Other files (download link)

---

## Search

Use the search bar to find:
- Messages by content
- Files shared
- Conversations with specific people

---

## Best Practices

1. **Use Channels Appropriately** - Keep discussions in relevant channels
2. **@mention Thoughtfully** - Avoid unnecessary notifications
3. **Thread Replies** - Keep conversations organized
4. **Check Before Posting** - Review for sensitive information
