# Admin Dashboard Guide

The Admin Dashboard provides centralized management for your SpokeStack organization.

---

## Accessing Admin

Navigate to **Admin** in the sidebar (requires Admin or Leadership role).

---

## Admin Sections

### Dashboard
Overview of your organization:
- Active users count
- Module usage statistics
- Recent activity log
- System health status

### Users
Manage team members:
- Add new users
- Edit roles and permissions
- Deactivate accounts
- Reset passwords

### Settings
Organization configuration:
- Company profile
- Branding (logo, colors)
- Default preferences
- Module enablement

### Integrations
Connect external services:
- Slack integration
- Calendar sync
- File storage
- API access

---

## Quick Actions

From the Admin Dashboard:

| Action | Description |
|--------|-------------|
| **Add User** | Invite new team member |
| **View Logs** | See activity history |
| **Export Data** | Download reports |
| **Manage Billing** | Subscription settings |

---

## Organization Settings

### General
- Organization name
- Primary contact
- Time zone
- Date format

### Branding
- Logo upload
- Primary color (default: #52EDC7)
- Secondary color
- Custom domain

### Security
- Password policies
- Session timeout
- Two-factor authentication
- IP allowlists

---

## Reports

Generate reports for:
- User activity
- Time tracking summaries
- Project budgets
- Leave balances

Export formats: CSV, PDF, Excel
