# User Management Guide

This guide covers adding, editing, and managing users in SpokeStack.

---

## Adding Users

### Single User
1. Navigate to **Admin** > **Users**
2. Click **Add User**
3. Enter user details:
   - Name
   - Email
   - Department
   - Role/Permission Level
4. Click **Send Invite**

User receives an email invitation to set up their account.

### Bulk Import
1. Navigate to **Admin** > **Users**
2. Click **Import Users**
3. Download the CSV template
4. Fill in user details
5. Upload the completed CSV
6. Review and confirm

---

## User Roles

| Role | Description | Typical Users |
|------|-------------|---------------|
| **Admin** | Full access, can manage all settings and users | IT, Founders |
| **Leadership** | Access to all modules + RFP | Directors, VPs |
| **Team Lead** | Can assign work, manage team | Department Heads |
| **Staff** | Standard employee access | Regular employees |
| **Freelancer** | Limited to assigned work | Contractors |
| **Client** | Client Portal access only | External clients |

---

## Editing Users

1. Find user in the user list
2. Click their name or the edit icon
3. Modify:
   - Profile information
   - Department assignment
   - Permission level
   - Module access
4. Save changes

---

## Deactivating Users

1. Find user in the user list
2. Click the more options menu (...)
3. Select **Deactivate**
4. Confirm the action

Deactivated users:
- Cannot log in
- Retain historical data
- Can be reactivated later

---

## Department Management

Organize users into departments:
- Creative
- Production
- Strategy
- Account Management
- Finance
- Operations

### Adding a Department
1. Go to **Admin** > **Settings** > **Departments**
2. Click **Add Department**
3. Enter name and optional lead
4. Save

---

## Weekly Capacity

Set user availability for resource planning:
- Default: 40 hours/week
- Adjust for part-time workers
- Account for administrative time

---

## User Profile Fields

| Field | Required | Description |
|-------|----------|-------------|
| Name | Yes | Full display name |
| Email | Yes | Login and notifications |
| Department | No | Organizational grouping |
| Role | Yes | Permission level |
| Title | No | Job title |
| Phone | No | Contact number |
| Avatar | No | Profile picture |
| Weekly Capacity | No | Hours available per week |
