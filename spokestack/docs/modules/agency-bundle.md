# Agency Bundle

The Agency Bundle provides professional services and client management tools.

---

## Modules

### 1. Clients

Client profiles and relationship management.

#### Features
- **Client Profiles** - Company information
- **Brand Guidelines** - Colors, fonts, assets
- **Contact Directory** - Key stakeholders
- **Activity History** - Interaction log

#### Client Fields
- Name
- Code (short identifier)
- Industry
- Primary Contact
- Brand Colors
- Logo
- Notes

---

### 2. Retainers

Monthly retainer tracking and allocation.

#### Features
- **Retainer Agreements** - Contract details
- **Hour Allocation** - Monthly budgets
- **Rollover Tracking** - Unused hours
- **Utilization Reports** - Budget vs. actual

#### Retainer Types
| Type | Description |
|------|-------------|
| Fixed | Set monthly fee |
| Hourly | Pay per hour |
| Hybrid | Base + hourly overflow |

---

### 3. Projects

Project-based work with budgets and timelines.

#### Features
- **List View** - All projects overview
- **Board View** - Kanban by status
- **Gantt View** - Timeline and dependencies

#### Project Fields
- Name
- Client
- Type (Retainer, Project, Pitch, Internal)
- Status
- Budget (hours/amount)
- Start/End Dates
- Team Members

#### Project Status
- Draft
- Active
- On Hold
- Completed
- Cancelled

---

### 4. Resources

Team capacity and workload management.

#### Features
- **Capacity View** - Team member availability
- **Workload View** - Current assignments
- **Timeline View** - Schedule visualization

#### Metrics
- Weekly Capacity (hours)
- Hours Logged
- Utilization %
- Active Briefs Count

#### Utilization Indicators
| Level | Color | Description |
|-------|-------|-------------|
| Over (100%+) | Red | Overallocated |
| High (80-99%) | Amber | Near capacity |
| Good (50-79%) | Green | Healthy workload |
| Low (<50%) | Blue | Available capacity |

---

### 5. CRM

Sales pipeline and contact management.

#### Features
- **Pipeline View** - Deal stages
- **Contact Management** - People and companies
- **Activity Tracking** - Calls, emails, meetings
- **Deal Forecasting** - Revenue projections

#### Pipeline Stages
1. Lead
2. Qualified
3. Proposal
4. Negotiation
5. Closed Won
6. Closed Lost

---

## Integration Points

The Agency Bundle integrates with:
- **Briefs** - Client work requests
- **Time Tracking** - Project hour logging
- **RFP** - New client pipeline

---

## Reports

Available reports:
- Client Revenue Summary
- Retainer Utilization
- Project Profitability
- Resource Allocation
- Pipeline Forecast
