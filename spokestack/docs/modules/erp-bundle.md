# ERP Bundle

The ERP Bundle provides core business operations for managing work, time, and team.

---

## Modules

### 1. Briefs

Work request management system with multiple views.

#### Features
- **List View** - Detailed table with sorting and filtering
- **Kanban View** - Visual workflow with drag-and-drop status
- **Timeline View** - Gantt-style deadline tracking

#### Brief Types
| Type | Icon | Description |
|------|------|-------------|
| Video Shoot | Camera | On-location video production |
| Video Edit | Film | Post-production editing |
| Design | Palette | Graphic design work |
| Copy (EN) | Pen | English copywriting |
| Copy (AR) | Pen | Arabic copywriting |
| Paid Media | Megaphone | Advertising campaigns |

#### Status Workflow
Draft → Submitted → In Progress → In Review → Completed

#### Priority Levels
- **Urgent** - Same-day turnaround
- **High** - 1-2 day turnaround
- **Medium** - Standard timeline
- **Low** - Flexible deadline

---

### 2. Time Tracking

Log and track billable and non-billable hours.

#### Features
- **Timer Mode** - Start/stop tracking
- **Manual Entry** - Add entries retroactively
- **Timesheets** - Weekly/monthly views
- **Reports** - Utilization and billing

#### Entry Fields
- Date
- Project/Client
- Brief (optional)
- Hours
- Description
- Billable toggle

---

### 3. Leave

Manage time-off requests and approvals.

#### Features
- **Request Leave** - Submit PTO requests
- **Calendar View** - See team availability
- **Approval Workflow** - Manager approvals
- **Balance Tracking** - Remaining days

#### Leave Types
- Annual Leave
- Sick Leave
- Personal Day
- Work From Home
- Unpaid Leave

---

### 4. Team

Employee directory and org structure.

#### Features
- **Directory** - Searchable team list
- **Org Chart** - Visual hierarchy
- **Profiles** - Detailed member info
- **Departments** - Grouped views

---

### 5. RFP

Request for Proposal pipeline management.

#### Features
- **Pipeline View** - Kanban-style stages
- **List View** - Detailed tracking
- **Timeline View** - Deadline visualization
- **Document Ingestion** - Drag & drop RFP docs

#### Pipeline Stages
1. Vetting
2. Active
3. Awaiting Review
4. Ready to Submit
5. Submitted
6. Awaiting Response
7. Won/Lost

#### RFP Form Fields
- Client/Entity Name
- Portal/Source
- Deadline
- Estimated Value
- Scope of Work
- Requirements
- Bid Bond Required
- Notes
- Attached Documents

---

## Best Practices

1. **Use Kanban for Daily Work** - Quick status updates
2. **Switch to Timeline for Planning** - See the big picture
3. **Log Time Daily** - Don't let entries pile up
4. **Request Leave Early** - Allow time for approvals
5. **Keep RFP Docs Attached** - Easy reference later
