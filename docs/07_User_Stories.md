## 7. User Journey: Brief Submission

```
Account Manager opens dashboard
         │
         ▼
Clicks "+ New Brief"
         │
         ▼
Selects brief type
         │
         ▼
Fills form (quality score updates)
         │
         ▼
Attaches files
         │
         ▼
Reviews AI suggestions
         │
         ▼
Clicks "Submit"
         │
         ▼
System: assigns number, notifies team lead
         │
         ▼
AM sees brief in "Submitted" status
```

---

## 8. User Journey: Work Assignment

```
Team Lead receives notification
         │
         ▼
Opens Team Lead Queue
         │
         ▼
Reviews brief details
         │
         ▼
Sees AI recommendation + match score
         │
         ▼
Checks assignee capacity
         │
         ▼
Clicks "Assign"
         │
         ▼
System: notifies assignee, updates status
```

---

## 9. User Journey: Work Completion

```
Creative opens dashboard
         │
         ▼
Sees assigned brief
         │
         ▼
Clicks "Start Work" (timer begins)
         │
         ▼
Works on deliverable
         │
         ▼
Stops timer, uploads deliverable
         │
         ▼
Clicks "Submit for Review"
         │
         ▼
Team Lead approves
         │
         ▼
Client portal updated
```
